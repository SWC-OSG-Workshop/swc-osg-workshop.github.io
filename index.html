General Information

The SWC-OSG-Workshop is a github organization. Under this organization, we have the course materials for the sofware carpentry and open science grid workshops.


general contains the basic learning modules and tools to generate the workshop website.


As soon as the workshop date is finalized, create a repo by the name "YYYY-MM-DD", where YYYY is the year, MM is the month and DD is the date of the workshop. In your local Desktop or Laptop, type the following git clone https://github.com/SWC-OSG-Workshop/YYYY-MM-DD.git cd YYYY-MM-DD git checkout -b gh-pages git remote add general https://github.com/SWC-OSG-Workshop/general.git git pull origin gh-pages git remote remove general Now follow the steps outlined in the README.md file to create the web-site for the workshop.
